### 🚀 Featured Engineering & AI Projects

#### 📦 **Q-Commerce Logistics Intelligence Engine**
*Python | XGBoost | Scikit-Learn | Groq API (Llama-3.1-8b) | Pandas*
> End-to-end ML pipeline streamlining logistics evaluation and delivery SLA predictions.
- **Impact:** Benchmarked XGBoost against baseline models to pivot strategy towards a GenAI-based Root Cause Analysis[cite: 1].
- **Architecture:** Architected a modular MLOps system, decoupling data processing, modeling, and inference logic for production-grade scalability[cite: 1].
[View Repository](https://github.com/L9G9N0/q-commerce-sla-predictor)

#### 🍏 **macOS FaceUnlock (Apple Silicon)**
*Python | C++ | MediaPipe | dlib | UNIX Sockets | PAM*
> An offline macOS authentication daemon utilizing 3D Liveness state machines.
- **Performance:** Achieved microsecond IPC latency by routing secure streams to the kernel via UNIX sockets and a custom C++ PAM module[cite: 1].
- **Optimization:** Accelerated verification to O(1) time complexity by caching 128-D facial vectors in RAM[cite: 1].
[View Repository](https://github.com/L9G9N0/MACOS-FACEUNLOCK)

#### ☁️ **OrbitSync Vault: Agentic Cloud Storage**
*FastAPI | PostgreSQL | Llama-3.1-8b | Watchdog | React*
> Cloud-native local sync daemon with automated semantic tagging.
- **AI Integration:** Engineered an Agentic AI pipeline for semantic tagging, optimizing file discoverability via PostgreSQL ILIKE vector-searches[cite: 1].
- **Core Engine:** Developed a daemon using Watchdog and FSEvents to achieve real-time data replication with zero manual delays[cite: 1].
[View Repository](https://github.com/L9G9N0/OrbitSync-Backend)

#### ⏳ **Devotee Mode - macOS Distraction Blocker**
*Swift | SwiftUI | AppKit | Combine Framework*
> Hardcore, native macOS menu-bar utility designed to force deep work.
- **Core System:** Engineered a "Sweep & Kill" engine using `NSWorkspace` to actively monitor and force-terminate blocked applications in real-time.
- **Architecture:** Bypassed standard window lifecycles to create a dock-hidden Menu Bar app using `NSApplicationDelegate` and `NSPopover`.
[View Repository](https://github.com/L9G9N0/Devotee-Mode-macOS)

#### 🧠 **TypeLearn AI: AI-Powered Learning Mentor**
*Next.js | Groq API | TypeScript | Llama-3.1-8b*
> Data-driven educational platform with real-time feedback systems.
- **AI Engine:** Leveraged Llama-3.1-8b for sub-second AI analysis to generate real-time feedback[cite: 1].
